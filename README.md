# API-escolar

API REST para **gestão escolar**, desenvolvida em **Node.js**, com foco em aprendizado de back-end, boas práticas e organização de código.

> Projeto em desenvolvimento

---

## Objetivo do Projeto

Criar uma API que permita gerenciar os principais elementos de uma escola, como:

- Alunos
- Professores
- Turmas
- Matérias
- Matrículas
- Notas

O projeto tem como foco praticar:
- Arquitetura MVC
- APIs REST
- Banco de dados relacional
- Relacionamentos entre entidades
- Versionamento com Git

---

## Tecnologias Utilizadas

- Node.js
- Express
- Banco de dados relacional PostgreSQL
- ORM Prisma
- Nodemon (ambiente de desenvolvimento)
- ESLint (padronização de código)
